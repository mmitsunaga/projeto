Configurando a Abertura do Assinador

Serão necessário 3 passos para habilitar o assinador externo no seu computador/navegador. Veja a seguir:

1 - Deixando o Navegador baixar automaticamente o Assinador

Temos que colocar que os arquivos JNLP devem ser abertos diretamente pelo JAVA no navegador firefox (ou outro)
No seu navegador encontre onde é definido a abertura do dos documentos do JNPL, veja a imagem abaixo.

2 - Baixe o S.A.I.

Clique aqui para baixar o Assinador, isso so será necessário nesse primeiro momento.
O navegador vai avisar que ele é um executável, mas não se procupe pois ele vai fazer somente as assinaturas. Assim clique em Ok.



Depois o Java vai perguntar: Deseja Executar este Aplicativo? Serviço de Assintura e Impressão(tjgo).
Verifique se o Editor: Tribunal de Justiça do Estado de Goias e Local: https://docs.tjgo.jus.br
Marque a opção: Não mostrar novamente para aplicativos do editor e local acima. Que essa mensagem não aparecerá novamente.


Depois de executar, o icone do S.A.I. vai aparecer perto do relógio do seu computador.

3 - Instalando o Certificado

Certificado

O Serviço de Assinatura e Impressão (S.A.I.) utiliza um certificado auto-assinado para fazer a comunicação entre o serviço local e o seu navegador.
Esse certificado garante a integridade e segurança dos dados enviados pelo seu navegador para o S.A.I.. Entretanto, o certificado auto-assinado não é reconhecido pelos navegadores, assim é necessária a instalação do mesmo.

Clique aqui para começar a instalar o certificado
O Navegador agora vai avisar que existe um potencial risco nesse certiticado, mas verifique que ele é para o endereço 127.0.0.1, ou seja, para uma comunicação interna, somente no seu computador, ele so garante que nenhum virus/intruso consiguirar ler o conteudo que sai do Assinador e chegar no Navegador. Clique em Avançado... e depois em Aceitar o risco e continuar


a mensagem abaixo aparecerá quando o procedimento estiver finalizado, agora o assinador já está habilitado para o seu navegador/computador.

Agora pode fechar essa aba e voltar para a tela de peticionamento.
