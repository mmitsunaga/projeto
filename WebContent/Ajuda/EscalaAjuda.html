Cadastro e manutenção de Escalas
     A criação de uma Escala é necessária para que os mandados judiciais sejam distribuídos para os oficiais de justiça.
     Cada escala é composta de uma fila de oficiais de justiça, e dados referentes ao tipo de mandado, área(criminal e cível) e a região a ser atendida pelo oficial. Esta região é composta por um bairro e zona. Ou seja, quando um mandado judicial de um determinado tipo, área e bairro é distribuído, este mandado é enviado para o próximo oficial de justiça da fila.
Cadastro e Alteração de Escala:
     Para cadastro de escala é necessário preencher todos os campos descritos abaixo:
  • Descrição: Uma breve descrição da escala.
  • Área: Área de atuação da escala.
  • Tipo de Mandado: Tipo único de mandado desta escala.
  • Zona: Zona de atuação deste tipo de mandado.
  • Região: Região geográfica.
  • Bairro: Bairro para distribuição do tipo de mandado.
  • Quantidade de Mandados: Quantidade de mandados que cada oficial pode receber.
  • Ativa: Se a escala ficará ativa ou inativa.
    •      Depois de preenchido estes campos basta clicar no botão Salvar
    Localizar Escala:
         Para localizar uma escala basta clicar no botão Localizar e consultar por nome ou navegar pela barra de navegação númerica logo abaixo da lista.
    Excluir Escala:
         Para Excluir uma escala, o campo Identificador deve estar preenchido. Pois este é o número que identifica a escala.
         Para Excluir você deve pesquisar e localizar a escala desejada. Em seguida clique no botão
